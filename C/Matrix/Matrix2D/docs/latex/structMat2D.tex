\item rows\+: number of rows (height)
\item cols\+: number of columns (width)
\item stride\+\_\+r\+: number of elements between successive rows in memory (for contiguous storage, stride\+\_\+r == cols)
\item elements\+: pointer to contiguous storage of size rows $\ast$ cols

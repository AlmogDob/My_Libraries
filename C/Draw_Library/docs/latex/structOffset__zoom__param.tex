\doxysubsection{Detailed Description}
Pan/zoom parameters relative to screen center. 

The coordinates are shifted by (offset\+\_\+x, offset\+\_\+y) and scaled by zoom\+\_\+multiplier about the screen center. The mouse fields are optional and can be used by UI code that updates the pan/zoom.

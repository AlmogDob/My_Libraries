\doxysubsection{Detailed Description}
A minor \char`\"{}view\char`\"{} into a reference matrix. 

Represents a minor by excluding one row and one column of a reference matrix. It holds index lists mapping into the reference matrix, without owning the data of the reference matrix itself.

Memory ownership\+:
\begin{DoxyItemize}
\item rows\+\_\+list and cols\+\_\+list are heap-\/allocated by minor allocators and must be freed with mat2\+D\+\_\+minor\+\_\+free.
\item The underlying matrix data (ref\+\_\+mat.\+elements) is not owned by the minor and must not be freed by the minor functions. 
\end{DoxyItemize}

\doxysubsection{Detailed Description}
Immediate-\/mode 2D/3D raster helpers for drawing onto \mbox{\hyperlink{structMat2D__uint32}{Mat2\+D\+\_\+uint32}} pixel buffers. 

This single-\/header library provides a minimal software rasterizer for drawing into a 32-\/bit ARGB pixel buffer (\mbox{\hyperlink{structMat2D__uint32}{Mat2\+D\+\_\+uint32}}). It supports\+:
\begin{DoxyItemize}
\item Points, lines, circles, triangles and quads (wire and filled)
\item Z-\/buffered triangle/quad rasterization (inverse-\/Z convention)
\item Per-\/vertex color and simple light-\/intensity interpolation
\item Basic vector-\/text drawing (ASCII subset)
\item Plotting helper types (\mbox{\hyperlink{structFigure}{Figure}}) and utilities for curve plots and 2D scalar-\/field visualization using perceptual color interpolation in the OKLab/\+OKLch color spaces
\item Cartesian grid generation in common planes
\end{DoxyItemize}

All draw calls may accept an \mbox{\hyperlink{structOffset__zoom__param}{Offset\+\_\+zoom\+\_\+param}} that enables simple pan/zoom behavior around the screen center.

Types \mbox{\hyperlink{structMat2D}{Mat2D}} and \mbox{\hyperlink{structMat2D__uint32}{Mat2\+D\+\_\+uint32}} are provided by \mbox{\hyperlink{Matrix2D_8h}{Matrix2\+D.\+h}}.

Usage\+:
\begin{DoxyItemize}
\item Include this header wherever you use the API.
\item In exactly one translation unit (source file) define ALMOG\+\_\+\+DRAW\+\_\+\+LIBRARY\+\_\+\+IMPLEMENTATION before including this header to compile the function definitions.
\end{DoxyItemize}

\begin{DoxyNote}{Note}

\begin{DoxyItemize}
\item Colors are ARGB in 0x\+AARRGGBB packed 32-\/bit format.
\item Z buffering uses an inverse-\/Z buffer (bigger is closer).
\item The OKLab/\+OKLch conversions here assume linear s\+RGB channels. 
\end{DoxyItemize}
\end{DoxyNote}

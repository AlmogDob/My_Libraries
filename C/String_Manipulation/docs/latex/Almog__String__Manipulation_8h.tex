\doxysubsection{Detailed Description}
Lightweight string and line manipulation helpers. 

This single-\/header module provides small utilities for working with C strings\+:
\begin{DoxyItemize}
\item Reading a single line from a FILE stream
\item Measuring string length
\item Extracting the next \char`\"{}word\char`\"{} (token) from a line using a separator
\item Cutting the extracted word from the source buffer
\item Copying a substring by indices
\item Counting occurrences of a substring
\item A boolean-\/style strncmp (returns 1 on equality, 0 otherwise)
\end{DoxyItemize}

Usage
\begin{DoxyItemize}
\item In exactly one translation unit, define ALMOG\+\_\+\+STRING\+\_\+\+MANIPULATION\+\_\+\+IMPLEMENTATION before including this header to compile the implementation.
\item In all other files, include the header without the macro to get declarations only.
\end{DoxyItemize}

Notes and limitations
\begin{DoxyItemize}
\item All destination buffers must be large enough; functions do not grow or allocate buffers.
\item asm\+\_\+get\+\_\+line enforces MAX\+\_\+\+LEN\+\_\+\+LINE characters (not counting the terminating \textquotesingle{}\textbackslash{}0\textquotesingle{}). Longer lines cause a fatal error via exit(1).
\item asm\+\_\+strncmp differs from the standard C strncmp\+: this version returns 1 if equal and 0 otherwise.
\item These functions are not locale-\/aware unless otherwise noted (isspace is used for whitespace handling). 
\end{DoxyItemize}

Extract the next word from a line without modifying the source. 

Skips leading whitespace in src (as determined by isspace), then copies characters into dst until one of the following is seen\+: the separator, a newline (\textquotesingle{}~\newline

Compare up to N characters for equality (boolean result). 

Returns 1 if the first N characters of s1 and s2 are all equal; otherwise returns 0. Unlike the standard C strncmp, which returns 0 on equality and a non-\/zero value on inequality/order, this function returns a boolean-\/like result (1 == equal, 0 == different).
